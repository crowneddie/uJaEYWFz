# 项目简介
牙科诊所预约管理系统 微信小程序+管理后台 支持预约挂号健康档案。核心功能：
• 用户端：预约挂号、健康档案、在线咨询、资讯阅读
• 管理端：用户管理、医生管理、科室管理、预约管理、就诊记录、数据统计


## 核心功能
- 用户端：
  - 预约挂号
  - 健康档案
  - 在线咨询
  - 资讯阅读
- 管理端：
  - 用户管理
  - 医生管理
  - 科室管理
  - 预约管理
  - 就诊记录
  - 数据统计

## 适用场景
- 牙科诊所预约挂号服务
- 患者健康档案管理
- 医生与患者在线沟通
- 诊所运营数据统计与分析

## 技术特点
- 微信小程序便捷接入
- 简洁直观的用户界面
- 强大的后台管理系统
- 数据安全性与隐私保护
- 多样化的功能模块
